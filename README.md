# UniHub Workshop

Đồ án TKPM (HCMUS) — hệ thống số hoá toàn bộ vòng đời workshop.

## Cấu trúc submission

```
UniHubWorkshop/
├── blueprint/   # Phần 1 — proposal.md, design.md, specs/* (OpenSpec)
└── src/         # Phần 2 — toàn bộ cài đặt (monorepo pnpm)
```

- **Phần 1 (Blueprint)** — xem `blueprint/README.md`.
- **Phần 2 (Cài đặt)** — xem `src/README.md` để build và chạy hệ thống.

> Mọi mã nguồn, Docker, scripts, dữ liệu mẫu đều nằm trong `src/`. Thư mục `blueprint/` chỉ chứa tài liệu thiết kế.
